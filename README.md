Для запуска тестов склоинируйте репозиторий и установите requirements при помощи bash команд:

git clone [https://github.com/Argentrix/TP_HW2.git](https://github.com/Argentrix/TP_HW2.git)
cd TP_HW2
py -m pip install -r requirements.txt
pytest
